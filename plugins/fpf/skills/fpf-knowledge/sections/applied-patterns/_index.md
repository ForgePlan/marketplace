# Applied Patterns

Practical, step-by-step guides that bridge the FPF specification and everyday use.
These patterns combine multiple FPF sections into workflows you can follow directly.

## Contents

- [Decomposition Guide](decomposition-guide.md) (54 lines) — How to decompose a system into bounded contexts with roles, responsibilities, and interfaces using FPF Part A.
- [Evaluation Guide](evaluation-guide.md) (61 lines) — How to evaluate decisions using F-G-R scoring and Trust Calculus from FPF Parts B and C.
- [Reasoning Guide](reasoning-guide.md) (60 lines) — How to use the ADI (Abduction-Deduction-Induction) reasoning cycle from FPF Part B.5.
- [Forgeplan Integration](forgeplan-integration.md) (63 lines) — How FPF outputs map to forgeplan artefacts: PRD=what, RFC=how, ADR=why, Evidence=trust.
