# Artifacts

Forgeplan uses typed artifacts to capture engineering decisions and track work. Each artifact type serves a specific purpose in the workflow.

## Artifact Types

| Type | When to use | Created by |
|------|------------|------------|
| PRD | Any Standard+ task that changes user-facing behavior | `forgeplan new prd` |
| RFC | Deep+ tasks requiring technical design discussion | `forgeplan new rfc` |
| ADR | Critical decisions with long-term consequences | `forgeplan new adr` |
| Evidence | After implementation to link code to requirements | `forgeplan new evidence` |

- [rfc.md](rfc.md) — Request for Comments
