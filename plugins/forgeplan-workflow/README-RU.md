# forgeplan-workflow

Плагин для Claude Code, обеспечивающий структурированную автоматизацию инженерного рабочего процесса для пользователей [forgeplan](https://github.com/forgeplan/forgeplan).

## Требования

- Установленный CLI Claude Code
- **forgeplan CLI** — приватное приложение ForgePlan. Для получения доступа обратитесь к администратору проекта. После получения: поместите бинарник в PATH и выполните `forgeplan init -y` в проекте.

## Установка

```bash
claude plugin add /path/to/forgeplan-workflow
```

Или клонируйте в директорию плагинов Claude Code.

## Возможности

### Команды

- **`/forge-cycle`** — Запуск полного инженерного цикла: проверка здоровья, маршрутизация, формирование, сборка, доказательства, активация, коммит. Проводит задачу от идеи до закоммиченного кода с полной трассируемостью.

- **`/forge-audit`** — Мультиэкспертный аудит кода. Запускает параллельных ревьюеров (логика, архитектура, безопасность, тесты, производительность, документация) и формирует структурированный отчёт с результатами.

### Агент

- **forge-advisor** — Фоновый советник, который предлагает маршрутизацию перед кодированием, доказательства после реализации и периодические проверки здоровья. Неблокирующий, все предложения опциональны.

### Хуки

- **Safety hook** — Блокирует опасные Bash-команды (force push, hard reset, rm -rf /, DROP TABLE).
- **PRD check** — Предупреждает при редактировании кода без активного PRD (требуется forgeplan).

### База знаний

- **forgeplan-methodology** — Agentic RAG скилл, покрывающий полную методологию forgeplan: этапы рабочего процесса, типы артефактов (PRD/RFC/ADR/Evidence), калибровка глубины, скоринг R_eff, quality gates и кросс-сессионная память.

## Обзор рабочего процесса

```
Route --> Shape --> Build --> Audit --> Evidence --> Activate
  |         |         |         |          |            |
  v         v         v         v          v            v
Depth    PRD/RFC   Code +    Review    Link to      Mark
Level    /ADR      Tests    findings   artifacts    complete
```
