# dev-toolkit

Универсальный инженерный инструментарий для Claude Code. Работает с любым проектом, любым языком.

## Что делает

- **Мультиэкспертный аудит** (`/audit`) — Запускает 4 параллельных агента-ревьюера (Логика, Архитектура, Безопасность, Тесты), которые анализируют ваш код и сообщают о находках по степени серьёзности, с автоисправлением критических проблем.
- **Спринты на основе волн** (`/sprint`) — Разбивает сложные задачи на параллельные волны выполнения, планирует подход, получает ваше одобрение, затем методично выполняет с отслеживанием прогресса.
- **Восстановление сессии** (`/recall`) — Восстанавливает контекст вашей сессии, читая CLAUDE.md, историю git и доступные инструменты памяти, давая краткий брифинг, чтобы вы знали, на чём остановились.

## Установка

```bash
claude plugin add /path/to/dev-toolkit
```

Или скопируйте папку `dev-toolkit/` в директорию плагинов Claude Code.

## Использование

### Аудит кода

```
/audit
```

Определяет язык и фреймворк вашего проекта, проверяет что изменилось (через git diff), затем запускает 4 параллельных экспертных ревью. Формирует структурированный отчёт с находками CRITICAL/HIGH/MEDIUM/LOW и предлагает автоисправление серьёзных проблем.

### Запуск спринта

```
/sprint Add user authentication with JWT tokens
```

Исследует контекст вашего проекта, разбивает задачу на 2-4 волны параллельной работы, представляет план на утверждение, затем выполняет волна за волной. Запускает тесты в конце и предоставляет полный отчёт.

### Восстановление контекста сессии

```
/recall
```

Читает CLAUDE.md, проверяет git status и последние коммиты, опрашивает доступные инструменты памяти и представляет брифинг на 5-10 строк, чтобы вы могли мгновенно вернуться в контекст.

## Хуки

Плагин включает два автоматических хука:

- **Safety hook** (PreToolUse на Bash): Блокирует опасные команды до их выполнения — `git push --force`, `git reset --hard`, `git clean -f`, `rm -rf /` и `DROP TABLE/DATABASE`. Защищает от случайного уничтожения данных.

- **Test reminder** (PostToolUse на Write/Edit): Когда редактирование файла добавляет новую публичную функцию или метод без теста, вы получаете однострочное напоминание о необходимости его написать.

## Агент

- **dev-advisor**: Фоновый советник, который предлагает запустить `/audit` после больших изменений, напоминает о тестах для новых функций, отмечает проблемы безопасности в коде аутентификации и рекомендует `/sprint` для сложных задач.

## Совместимость

Работает с: JavaScript/TypeScript, Python, Rust, Go, Java/Kotlin, Ruby, PHP, C#/.NET и любым другим языком со стандартными конвенциями проекта.
