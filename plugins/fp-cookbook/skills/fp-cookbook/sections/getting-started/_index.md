# getting-started

Recipes for new forgeplan users: install the CLI, initialise a workspace,
and walk through a first PRD from create to activate.

## Contents

| File | Description | Lines |
|------|-------------|-------|
| [install-forgeplan-cli.md](install-forgeplan-cli.md) | Install via cargo or brew, verify version, first `forgeplan health` | 48 |
| [init-workspace.md](init-workspace.md) | `forgeplan init` in a repo — what files are created, .gitignore patterns | 52 |
| [first-prd-walkthrough.md](first-prd-walkthrough.md) | End-to-end: new PRD → validate → activate with live PRD-038 as example | 65 |

## Quick orientation

```
Install → forgeplan init → forgeplan new prd → validate → activate
```

All three recipes together take ~10 minutes on a fresh machine.
After `first-prd-walkthrough.md` you will have shipped your first real artifact.
