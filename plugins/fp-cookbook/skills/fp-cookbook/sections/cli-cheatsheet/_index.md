# cli-cheatsheet

One-page quick references for every forgeplan CLI command and every
`mcp__forgeplan__*` MCP tool. Load only the file you need.

## Contents

| File | Description | Lines |
|------|-------------|-------|
| [cli-quick-ref.md](cli-quick-ref.md) | Every `forgeplan <command>` with synopsis and example — one-liner format | 75 |
| [mcp-quick-ref.md](mcp-quick-ref.md) | Every `mcp__forgeplan__forgeplan_*` MCP tool with parameters + canonical example | 80 |

## When to use

- Forgot the exact flag for `forgeplan validate`? → `cli-quick-ref.md`
- Need the parameter shape for `forgeplan_dispatch`? → `mcp-quick-ref.md`
- Conceptual question (what is R_eff)? → use `forgeplan-methodology` skill instead

## Format

Both files use compact tables: command/tool name, key parameters, one-line example.
They are NOT exhaustive API docs — for full docs see the forgeplan repo.
