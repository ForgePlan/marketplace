## Out-of-scope archive

Rejected ideas live in `.out-of-scope/{topic}.md`. The `triage` skill reads
them before grilling, to avoid re-suggesting things you already declined.

Template entry:

```markdown
# {Topic}

**Date**: YYYY-MM-DD
**Status**: rejected
**Why**: {one paragraph explanation}
**Re-evaluate when**: {condition that would change the answer, or "never"}
```
